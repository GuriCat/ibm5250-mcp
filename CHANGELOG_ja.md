# 変更履歴

IBM 5250 MCP サーバー配布物の利用者向け変更点を、日付の新しい順にまとめます。

> English version: see [`CHANGELOG.md`](CHANGELOG.md).

期間限定ベータ版です。**Windows (x64) のみ。2026-07-31 まで有効。** ソースコードは非公開です。
検証は日本語環境のみ（一次言語 2962、QCCSID 65535）。

---

## 2026-07-01

### 追加
- **セッションごとに独立したビューアーウィンドウ／ポート** — 接続のたびに専用のポートとブラウザー窓を割り当て。ACS の「同じ構成で実行」に相当し、同一構成の接続を複数同時に開けます
- **接続時のビューアー自動表示** — `config.json` の `viewer.auto_open` で制御（`off` ／ 通常ブラウザー窓 ／ アプリ窓。既定はアプリ窓）
- **ウィンドウモード切替ボタン**（ツールバー）— アプリ窓 ⇄ 通常ブラウザー窓を切り替え。押した窓をその場で開き直し、次回起動時の既定も同時に変わります

### 変更
- **切断とウィンドウの双方向連動** — MCP から切断するとそのビューアー窓が閉じ、ビューアー窓を閉じると数秒後にセッションが自動的に切断されます
- **`session_id` の自動生成** — 接続時に自動採番（明示指定は廃止）

## 2026-06-28

### 変更
- **英語 UI の新規接続の既定を非日本語向けに変更** — 英語 UI を選ぶと、新規接続構成の既定がホストコードページ cp037・フォント Consolas になります（日本語 UI は従来どおり cp1399・ＭＳゴシック）。UI 言語を切り替えると、この既定も切り替わります
- **ビューアーのポート設定を一本化** — `viewer.port_default` に統一（プリセット単位の viewer_port を廃止）

### 追加
- **英語版マニュアルの 5250 画面を英語化** — サインオン・主メニュー・WRKACTJOB などの画面例を英語表示に差し替え

## 2026-06-21

### 修正
- **入力欄の下線ずれを修正** — 行末に全角文字が続く一部の画面で、入力欄の下線が右に 2 桁ずれて表示される問題を修正

### ドキュメント
- **管理ガイド** — ビューアーを Playwright と併用する際の手順・注意点を追記

## 2026-06-15

### 修正
- **高速入力の取りこぼしを解消** — ビューアーへの入力を直列化し、速く打鍵したときの文字の取りこぼしや順序の乱れをなくしました

## 2026-06-14 — 基盤機能

この日時点での配布物の主な機能です。以降の変更は上に日付降順で記載します。

### 端末エミュレーション
- TN5250E プロトコル、EBCDIC SBCS / DBCS 対応（cp1399、cp5026_current、cp5026_legacy、cp037、cp5035 など）
- 画面サイズ 24×80（DS3）と 27×132（DS4）。端末タイプは画面サイズ + コードページから自動選択
- 表意文字（DBCS）フィールドタイプ J / E / G / O に対応（SO/SI 制御、IME 日本語入力）
- フィールド処理: 数値 / 符号付き数値、フィールドイグジット（負符号含む）、モノケース、必須 / 自動実行、セグメント化された CNTFLD チェーン、ワードラップ

### Web 画面ビューアー（`http://localhost:5250`）
- 色属性、カーソル、OIA（操作情報行）に `X II` キーボードロック・`COMM659`（切断）表示
- 表示フォント / サイズ / 太字 / 斜体を設定可能。言語別の既定（日本語: ＭＳゴシック、英語: Consolas）
- ファンクションキーツールバー（F1〜F24）、PRINT、SysReq、Reset
- 貼り付けの取り消し（Ctrl+Z）、矩形貼り付けのあふれ（下の行へ流し込み）、スクロールバー操作
- 音声入力（認識言語は UI 言語に連動）
- ツールバー非表示、入力専有（排他 / 共有）モードによる複数クライアント閲覧
- UI 言語切替（日本語 / 英語）

### 運用
- `config.json` の接続プリセット、サインオン自動化の AUTOTYP スクリプト
- 環境変数によるヘッドレスパスワード、セッションログ
- 自己完結の単一実行ファイル（ランタイム同梱、Node.js 不要）。`config.json`・プリセット・ログは実行ファイルを置いたフォルダーから読み書き

### ドキュメント
- 英語・日本語の**ユーザーガイド**・**管理ガイド**（PDF）を `docs/manuals/` に同梱
- AI エージェントから操作するための `AI-GUIDE.md`

### この版での主な改善
- マニュアルの訂正（シフト文字の表示 / 非表示、音声入力の自動オフ、`X SYSTEM` の SysReq 操作、SBCS コード表 5 種の追加）
- 音声入力の認識言語を UI 言語（日本語 / 英語）に自動連動
- ツールバーアイコンを実ボタンの画像に差し替え、カーソル表示・OIA 挿入マークの記述を実装に合わせて訂正
